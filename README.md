# FreelanceHub

## О проекте

FreelanceHub — это современная веб-платформа для взаимодействия заказчиков и фрилансеров. Система позволяет размещать проекты, искать исполнителей, отправлять отклики, управлять заказами и вести коммуникацию между участниками платформы.

Проект разработан как полноценное клиент-серверное приложение с использованием современных технологий веб-разработки и контейнеризации.

---

## Основные возможности

### Для заказчиков

* Регистрация и авторизация
* Создание и редактирование профиля
* Публикация проектов
* Управление заказами
* Просмотр откликов от фрилансеров
* Выбор исполнителя
* Чат после выбора
* Оценка после оплаты 
* Отслеживание статуса выполнения проекта

### Для фрилансеров

* Регистрация и авторизация
* Создание профессионального профиля
* Поиск доступных проектов
* Отправка откликов
* Управление собственными заявками
* Работа с назначенными заказами
* Оценка после получение оплаты

### Общие возможности

* JWT-аутентификация
* Ролевая модель пользователей
* Работа с профилями
* REST API
* Валидация данных
* Контейнеризация через Docker
* Работа с PostgreSQL

## Запуск проекта через Docker

### Клонирование репозитория

```bash
git clone <repository-url>
cd FreelanceHub
```

### Создание файла окружения

Создать файл:

```env
backend/.env
```

Пример содержимого:

```env
DATABASE_URL=postgresql://user:password@db:5432/freelancehub
SECRET_KEY=your_secret_key
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30
```

### Запуск контейнеров

```bash
docker compose up --build
```
